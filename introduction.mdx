## Welcome to PPS Proyecto JMMR

PPS Proyecto JMMR is a lightweight authentication frontend that provides clean, modern login and registration forms. Built with pure HTML and styled with Tailwind CSS, this project demonstrates fundamental web development concepts with a focus on user experience and responsive design.

## Project Structure

The project consists of two main HTML pages located in the `frontend` directory:

- **login.html** - User authentication interface
- **registro.html** - New user registration form

Both pages share a consistent design language with:
- Centered card-based layouts
- Form validation support
- Hover states and transitions
- Accessible form controls
